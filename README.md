# cse165-adventuremachinevr

![Demo](demo.gif)

## How to Play

Go to releases and download the zip build. Unzip then run `adventuremachinevr.exe`

<b>Only works on Oculus Rift. There are no menus so to close the game just press Alt+F4</b>
